## Description

Which key opens your roof box? Which one fits the bike rack? Stop guessing.

Thule Key Tracker is a simple, offline utility for anyone who owns Thule lockable accessories. Log the key code for each product — roof boxes, bike racks, ski carriers, roof bars — and instantly look up which key opens what.

**THE PROBLEM**
Thule products use a universal numbered key system (N001–N250), but when you own several products, the keys all look the same. You end up in a car park, trying every key on every lock, or digging through drawers for that sticky note you wrote the number on months ago.

**THE SOLUTION**
Open the app. See every product and its key code at a glance. Done.

**KEY FEATURES**
• Catalogue your Thule gear — roof boxes, bike racks, ski carriers, roof bars, kayak carriers, cargo carriers, and more
• Large, prominent key codes on every screen — readable at arm's length in dim car park lighting
• Search by key code, product name, or type — find the right key in seconds
• Group by key code — perfect if you've upgraded to the Thule One-Key System
• Duplicate entries — quickly add multiple products sharing the same key
• Share key codes — send "Roof box — Key N125" to your partner in one tap
• Export & import — back up your data as JSON, restore on a new device
• Works 100% offline — no account, no server, no internet required
• Supports Dark Mode — designed for dark mode first

**PRIVACY**
Your data never leaves your device. No account. No analytics. No tracking. Pure utility.

## Review Notes for Apple

This is a simple offline utility app for tracking Thule lock key codes. It uses SwiftData for local-only persistence and makes no network requests. There is no account system, no in-app purchases, and no third-party SDKs. The app is not affiliated with Thule Group — it is an independent tool for Thule product owners.

All data is stored on-device only. The export/import feature uses iOS file picker to save/load JSON files locally.
